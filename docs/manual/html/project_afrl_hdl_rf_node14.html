sdcard

contains images for sdcards. It will contain folders with the target name, and within it are the parts used for the image and a final image, sdcard.img. The sdcard.img file is the one used with your sdcard imaging software to put it on its destination sdcard. BOOTFS contains all of the boot files from the build processes and rootfs contains the image for the base file system in ext4.
