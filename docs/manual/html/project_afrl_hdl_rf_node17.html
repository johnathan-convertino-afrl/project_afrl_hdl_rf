FMCOMMS2-3

The FMCOMMS2-3 is a Analog Devices FMC development board for VHF and UHF ranges. This project uses the Analog Devices HDL base project for the FPGA. This base has customisation's by AFRL for the Arria10 based targets. These targets do not exist in the original. It also fixes a few bugs and makes the data path for RX/TX a mirror image of each other.
